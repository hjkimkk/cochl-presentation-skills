# Pitch Deck Skill — Usage

## Triggers
- `/pitch-deck-skill` — map new content into the Cochl PT template
- `/cochl-pitch-deck` — build a production-grade pitch deck from scratch

## Assets
- `assets/Cochl_PT_template.pptx` — official Cochl PowerPoint template (22 slides, 10×5.625in)

## Key Arguments
| Arg | Description |
|---|---|
| `product` | Product or company name |
| `audience` | investors · government · enterprise · partner |
| `tagline` | One-line value proposition |
| `output` | Output file path |

## Output
- Self-contained HTML (navigable, keyboard arrows)
- PPTX via `scripts/gen_pptx.py`

See `examples/` for reference decks.
